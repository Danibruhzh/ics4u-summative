# :movie_camera: [TMDB Movie Rental App](https://freakflix.vercel.app)

A fake movie rental web application using the [TMDB API](https://www.themoviedb.org/documentation/api), built with **React**, **Firebase**, and **Firestore**. Users can browse movies, add them to a cart, and simulate a purchase flow.

---

## :rocket: Features

- :mag: Browse real movies via TMDB API
- :shopping_cart: Add movies to cart
- :credit_card: Simulate a purchase workflow
- :closed_lock_with_key: User authentication with Firebase Auth
- :cloud: Firestore integration for storing user/cart data

---

## :toolbox: Tech Stack

- **Frontend:** React, React Router, Axios
- **Backend:** Firebase Auth, Firestore
- **API:** [TMDB API](https://www.themoviedb.org/documentation/api)
